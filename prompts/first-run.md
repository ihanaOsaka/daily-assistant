# 初回起動プロンプト
# 以下をClaude Codeにコピペしてください

Daily Assistantの初回セットアップをお願いします。

1. まず /mcp でMCPサーバーの接続状態を確認してください
2. Google Workspace MCPサーバーが接続されていたら、Google Tasks のタスクリスト一覧を取得してOAuth認証をテストしてください
3. 認証が通ったら、Google Calendar（今日の予定）と Gmail（未読メール数）もテストしてください
4. 全サービスのテスト結果をまとめて報告してください

※初回はブラウザでGoogleアカウント認証が必要です。「このアプリはGoogleで確認されていません」と出たら「詳細」→「安全ではないページに移動」で進めてOKです。
