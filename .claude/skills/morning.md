朝のルーティンを実行してください。以下の手順で進めてください:

1. **今日のスケジュール確認**: Google Calendar から今日の予定を取得して表示
2. **タスク一覧確認**: Google Tasks から未完了のタスクを取得して表示
3. **未読メール確認**: Gmail から重要な未読メールがあれば要約して表示
4. **今日の計画提案**: 上記を踏まえて、今日の優先順位と推奨スケジュールを提案

表示フォーマット:
```
## おはようございます！今日は [日付] です

### 今日のスケジュール
- [時間] [予定名]
- ...

### 未完了タスク（優先度順）
- [ ] [タスク名] (期限: [日付])
- ...

### 重要なメール
- [差出人]: [件名] - [要約]
- ...

### 今日のおすすめアクション
1. [提案]
2. ...
```
